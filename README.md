# GUARDIAN xMS

> A modular platform for monitoring, managing and automating technical systems.

## Vision

GUARDIAN xMS is an open, vendor-neutral platform for monitoring, analysing and operating technical infrastructures.

The project started with monitoring a residential energy storage system, but is intentionally designed as a generic platform for technical assets, devices, sensors, actuators, events, incidents, health models and automation.

## Principles

- Architecture before implementation
- GitHub is the Single Source of Truth
- Domain Driven Design
- Vendor-neutral core
- Recommendation before automation
- Explainable health scores
- Security by design
