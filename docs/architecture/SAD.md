# Software Architecture Document (SAD)

**Projekt:** GUARDIAN xMS

## Status

🚧 In Bearbeitung (Living Document)

## Zweck

Das SAD beschreibt die Gesamtarchitektur von GUARDIAN xMS und verweist auf die einzelnen Detaildokumente und ADRs.

## Architekturprinzipien

- Architecture before implementation
- GitHub is the Single Source of Truth
- Domain Driven Design
- Vendor-neutral core
- Recommendation before automation
- Explainable Health Scores
- Security by Design
